Profil

Saya adalah mahasiswa Sistem Informasi di Universitas Trunojoyo Madura dengan minat yang mendalam dalam pengembangan perangkat lunak, teknologi informasi, dan analisis data. Saat ini, saya aktif sebagai asisten praktikum di kampus, di mana saya tidak hanya berperan dalam membantu mahasiswa lain memahami materi praktikum, tetapi juga mengembangkan keterampilan saya dalam mengajar dan berkomunikasi secara efektif. Selain itu, saya memiliki pengalaman dalam tata kelola teknologi informasi, khususnya dalam penerapan framework COBIT 2019 untuk meningkatkan efektivitas dan efisiensi layanan di berbagai organisasi.

Dalam bidang akademik dan penelitian, saya tertarik pada berbagai topik seperti deep learning, audit teknologi informasi, dan pengembangan sistem rekomendasi berbasis data. Saya juga tengah mengerjakan proyek sistem forecasting pengunjung Candi Borobudur menggunakan framework Laravel 9, di mana saya berusaha menerapkan metode prediksi yang akurat untuk membantu pengelola candi dalam mengoptimalkan layanan pengunjung. Proyek ini tidak hanya mengasah keterampilan teknis saya dalam pengembangan aplikasi web, tetapi juga memberikan pengalaman praktis dalam mengolah data dan menerapkan logika bisnis dalam sistem.

Saya memiliki visi jangka panjang untuk melanjutkan studi S2 di Universitas Indonesia, sebagai bagian dari upaya saya untuk memperdalam pengetahuan dan memperluas wawasan dalam bidang teknologi informasi. Saya percaya bahwa pendidikan lanjutan akan memberikan peluang lebih besar untuk berkontribusi dalam pengembangan teknologi dan membawa dampak positif bagi masyarakat.

Di luar bidang akademik, saya juga fokus pada pengembangan soft skill, termasuk public speaking, komunikasi, dan kemampuan pemecahan masalah. Saya secara aktif mencari peluang untuk melatih keterampilan ini, baik melalui kegiatan organisasi, proyek kolaboratif, maupun kesempatan berbicara di depan umum. Saya juga memiliki komitmen untuk terus meningkatkan kemampuan dalam matematika, bahasa Inggris sesuai standar TOEFL, dan informatika. Saya menerapkan latihan harian sebagai bagian dari rutinitas saya, dengan alokasi waktu 2-3 jam setiap harinya untuk memastikan perkembangan yang konsisten dan terarah.

Secara keseluruhan, saya berusaha untuk selalu berkembang, baik dalam aspek teknis maupun personal. Saya yakin bahwa dengan kombinasi antara pengetahuan akademik, keterampilan praktis, dan soft skill yang kuat, saya dapat mencapai tujuan akademik dan profesional saya, serta memberikan kontribusi nyata dalam dunia teknologi informasi.
